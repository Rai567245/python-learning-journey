[PYTHON-STRING-CONCATENATION] - To cancatenate, or combine, two strings you can use the + operator.

(1) Example

Merge varibale [a] with variable [b] into variable [c]:

>>>a = "Hello"
>>>b = "World"
>>>c = a + b
>>>print(c)

(2) Example

To add space between them, add a " ":

>>>a = "Hello"
>>>b = "World"
>>>c = a + " " + b
>>>print(c)
